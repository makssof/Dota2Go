# Dota2Go

Auto-Accepter for Dota2 Games

## Requirements

  - Go
  - GCC

## Building

```sh
$ make build
```

It will compile into `bin/` folder.

>Please make sure that you have `scheme` file near application. The `make` command makes it for you.
